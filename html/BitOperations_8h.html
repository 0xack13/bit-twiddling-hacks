Detailed Description

Several bit operations and manipulations, also known as 'bit hacks'.

Note
Some functions below are based on Sean Eron Anderson's Bit Twiddling Hacks. Those functions are shown in the table below with links to their documentation.
Function Documentation
isPositive Compute the sign of an integer
haveOppositeSigns Detect if two integers have opposite signs
min Compute the minimum (min) or maximum (max) of two integers without branching
max Compute the minimum (min) or maximum (max) of two integers without branching
isPowerOf2 Determining if an integer is a power of 2
modifyBits Conditionally set or clear bits without branching
mergeBits Merge bits from two values according to a mask
nBitsSet Counting bits set in 14, 24, or 32-bit words using 64-bit instructions
isOddParity Compute parity of word with a multiply
isEvenParity Compute parity of word with a multiply
reverseBitOrderByte Reverse the bits in a byte with 3 operations (64-bit multiply and modulus division)
reverseBitOrder Reverse an N-bit quantity in parallel in 5 * lg(N) operations
roundUpToPowerOf2 Round up to the next highest power of 2 by float casting
